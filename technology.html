Scalable, repeatable, cloud identity solution

Cloud Technology

We use the Amazon Cloud to provide scale for our Identity Service, and Ansible to provision the hardware and software. This provides scale and a repeatable, reliable way create environments.

Repeatable Deployment

Ansible creates a secure network environment, virtual servers and deploys the software to build a working forgerock identity environment. This allows us to upgrade with minimal downtime and ensure that all our environments are the same.

Secure Key Management

Keys and passwords are created by the deployment process and locked in an encrypted data store that is only accessible by the ForgeRock servers. This approach minimises the risks of secure key data being accessed by unauthorised users and locks down access onlyo to a small set of administrators.

Manageable

The environments are managed using centralised logging, operational intelligence and alerting. We can provide office hours, or 24x7 support.
